# RoyalChain Casino Backend

## Overview
Backend API for RoyalChain Casino web application.

## Tech Stack
- Node.js
- Express
- MongoDB Atlas
- JWT
- Mongoose

## Setup
npm install
node app.js

## Environment Variables
MONGO_URI=
JWT_SECRET=
PORT=

## API Endpoints
POST /api/auth/register
POST /api/auth/login
GET /api/users/profile
PUT /api/users/profile
POST /api/games/coinflip
POST /api/games/roulette
